# 이슈 003: 공개용 kakaocli를 운영자용 agent 레포에서 분리

**상태**: 완료
**생성일**: 2026-03-08

## 배경

현재 레포는 운영자 전용 agent 지침과 공개용 `kakaocli` 제품 코드를 함께
가지고 있습니다. 목표 상태는 이 레포를 운영자용 비서 작업공간으로만
남기고, 공개 제품은 별도 로컬 저장소로 분리하는 것입니다.

## 완료 기준

- [x] 공개용 `kakaocli` 스냅샷이 이 작업공간 밖의 별도 로컬 저장소에 존재한다.
- [x] 이 레포에는 운영자 전용 지침, 문서, 얇은 래퍼만 남는다.
- [x] 제품 코드와 제품 테스트/설정 디렉터리가 이 레포에서 제거된다.
- [x] `AGENTS.md`와 `.agents/skills/kakaocli/SKILL.md`가 외부 `PATH` 명령 사용과 미설치 시 조치를 안내한다.

## 작업 목록

- [x] 1. 현재 공개 준비 트리를 별도 로컬 저장소로 내보낸다.
- [x] 2. 운영자 지침과 루트 래퍼를 외부 `PATH` 명령 호출 방식으로 바꾼다.
- [x] 3. 내장된 공개 제품 코드를 이 레포에서 제거한다.
- [x] 4. 래퍼와 지침을 검증하고 한국어 커밋 메시지로 기록한다.

## 참고 사항

운영자용 레포는 agent 동작 지침의 기준 저장소로 남겨야 한다.
공개용 레포는 별도로 관리하며, 이 작업공간 안에 브랜치나 내장 디렉터리
형태로 남기지 않는다.

정리했다. 내장 공개 제품 코드는 제거했고, 이 작업공간에서는
